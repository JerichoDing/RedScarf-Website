案例概述

申请类型： 本科转学

背景成绩：GPA 4.0 TOEFL 106

录取结果：加州大学圣塔芭芭拉分校 UCSB 罗切斯特大学 UR 加州大学圣地亚哥分校 UCSD 波士顿大学 BU 雪城大学

录取专业：经济与会计，金融，商业管理

优劣分析
学生优势：1. 学生成绩很好，GPA4.0，托福106。2. 各种材料齐全，有一定课外活动经历。
学生劣势：在ASU修的calculus和accounting课程名字比较奇怪，不确定申请的学校是否认可。
专家分析
C同学高中毕业后先在某大学读了一年类似预科的项目，然后转入某商学院学习，专业是Business Data Analytics。成绩保持非常好，几乎所有课都是A或A+，但有些课程名称很奇怪，很难判定新的学校是否认可。C同学希望继续读business analytics或近似的方向。 C同学修的calculus和accounting课程名字比较奇怪，不确定申请的学校是否认可。 C同学希望继续读business analytics或近似方向，但UC本科几乎不开商科专业，所以建议C同学考虑比较接近的business economics，其它排名稍低的申请了商学院的business analytics或者information system。由于NYU难度较高，红领巾团队老师和C同学沟通后不建议申商学院，最终以economics申请。由于Rochester商学院对于转学分要求很严格，所以申请了文理学院financial economics专业。红领巾团队老师建议C同学选校和专业时，考虑商学院通常难度较高、竞争激烈、先修课要求严格、转学可能受限的问题。而C同学修过较多名称奇怪或方向较细的专业课，很可能不被其他商学院接受。鉴于C同学比较在乎转学分数量，想尽快毕业，所以最终选择比较平衡，排名较高的学校基本申请了文理学院的项目，排名相对低的申请了商学院。 最后C同学选择了入读UCSD。红领巾团队祝福C同学在今后的留学道路上一帆风顺！
