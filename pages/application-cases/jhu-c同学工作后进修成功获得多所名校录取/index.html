案例概述

申请类型： 研究生申请

背景成绩：GPA 2.66

录取结果：约翰霍普金斯大学 JHU 康奈尔大学 Cornell 北卡罗来纳大学教堂山分校 东北大学

录取专业：MBA，EMBA

优劣分析
学生优势：1.工作经验丰富，2004年开始工作了。2.跨国工作经历丰富，从中国-新加坡-德国-美国，非常国际化的经历（虽然都是同一个公司）。3.绿卡身份，语言不需要考。
学生劣势：1.国内本科，GPA较低，做了WES认证以后，只有2.66，比学校成绩单上的成绩还低一些。2.没有在美国就读degree的经历。
专家分析
C同学本科就读于国内某大学，GPA 2.8，已工作多年，在美工作已获得绿卡身份，从2011年赴美工作以后，就一直在现在的公司从事半导体行业的工作。C同学在工作中遇到瓶颈期，希望能够提升管理知识，往管理岗位更进一步进行提升。因此希望借助疫情下，大部分院校不需要GRE的契机，申请MBA项目，获得研究生硕士学历，同时提升自己的管理知识。 C同学工作经验丰富，拥有绿卡身份，来自于现在热门的半导体行业。虽然本科GPA低，但是希望能够拿到一个顶尖院校的offer去就读MBA项目。 厚仁团队老师提供了合理的院校安排，确保满足C同学需求同时保证申请结果：冲刺院校为主（康奈尔，哈佛，JHU），1所东北大学作为保底。厚仁团队老师结合行业的特殊性，全球化的工作经历背景，工作职能、角色的转变等方面，整理了MBA优秀文书范例给C同学参考。文书老师、申请老师与C同学进行头脑风暴，draft后多次meeting修改润色，并提供了额外文书解释GPA问题。C同学一开始的简历有很多晦涩难懂的行业的术语，文书老师和申请老师一起帮助C同学精炼简历内容，突出重点，让招生官能够更加好的理解。 最终C同学收获了Cornell，UNC，JHU，Northeastern的全部院校的录取，专门写了感谢信发给了厚仁团队老师。厚仁团队祝福C同学未来可期！
